CSS Colors

CSS Color Names

style="color:Tomato;"

Note: CSS/HTML support 140 standard color names.

RGB - rgb(red, green, blue)

Each parameter defines the intensity of the color between 0 and 255 (decimal).

style="background-color:rgb(255, 99, 71);"

Note: Shades of gray are often defined using equal values for all the 3 light sources.

RGBA - rgba(red, green, blue, alpha)

Note: The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all) and it specifies the opacity for a color.

style="background-color:rgba(255, 99, 71, 0.5);"

HEX - #rrggbb

rr (red), gg (green) and bb (blue) are hexadecimal values between 00 and ff (same as decimal 0-255).

style="background-color:#ff6644;"

Note: Shades of gray are often defined using equal values for all the 3 light sources.

3 Digit HEX Value (#rgb)

The 3-digit hex code can only be used when both the values (RR, GG, and BB) are the same for each components.

style="background-color:#f64;"

HSL - hsl(hue, saturation%, lightness%)

style="background-color:hsl(9, 100%, 64%);"

Hue is a degree on the color wheel from 0 to 360. 0 is red, 120 is green, and 240 is blue.

Saturation is a percentage value, 0% means a shade of gray, and 100% is the full color. It can be described as the intensity of a color.

100% is pure color, no shades of gray.

50% is 50% gray, but you can still see the color.

0% is completely gray, you can no longer see the color.

Lightness is also a percentage, 0% is black, 50% is neither light or dark, 100% is white. It can be described as how much light you want to give the color.

0% means no light (black).

50% means 50% light (neither dark nor light).

100% means full lightness (white).

Note: Shades of gray are often defined by setting the hue and saturation to 0, and adjust the lightness from 0% to 100% to get darker/lighter shades.

HSLA - hsla(hue, saturation%, lightness%, alpha)

Note: The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all) and it specifies the opacity for a color.

style="background-color:hsla(9, 100%, 64%, 0.5);"
