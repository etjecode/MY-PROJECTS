The onclick Event

An onclick attribute (with code), is added to a button element to change the content of the "p" element (id="demo") below it:

onclick="document.getElementById('demo').innerHTML=Date()"




The code changes the content of its own element (using this.innerHTML):

onclick="this.innerHTML='YOU CLICKED ME!'"

Click me.



onclick="this.innerHTML=Date()"



Event attributes can call functions to execute JS code:

onclick="myFunction()"

The onclick event triggers a function when an element is clicked on.

Click to trigger a function that will output "Hello World" as the content of the "p" element (id="demo3") below it:

Function is defined in the script element.

function myFunction() { document.getElementById("demo3").innerHTML = "Hello World"; }

Use the addEventListener() method to attach a "click" event to a "p" element (id="demo6"):

document.getElementById("demo6").addEventListener("click", clickFunction);
function clickFunction() { this.innerHTML = "YOU CLICKED ME!"; }

Use the same method to change the content of the "input" element below the "p" element:

document.getElementById("demo7").addEventListener("click", clickFunction);
function clickFunction() { document.getElementById("demo8").value = "Output"; }
